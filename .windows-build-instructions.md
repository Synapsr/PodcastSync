# Instructions pour builder sur Windows

Si vous avez accès à une machine Windows, voici comment créer le build :

## Prérequis Windows

1. **Visual Studio Build Tools** : https://visualstudio.microsoft.com/downloads/
   - Sélectionnez "Desktop development with C++"

2. **Rust** : https://rustup.rs/
   ```powershell
   winget install Rustlang.Rustup
   ```

3. **Node.js** : https://nodejs.org/ (version 18+)

4. **WebView2** (normalement déjà installé sur Windows 10/11)

## Build Windows

```powershell
# Cloner le repo
git clone https://github.com/Synapsr/PodcastSync.git
cd PodcastSync

# Installer les dépendances
npm install

# Build production
npm run tauri build
```

## Fichiers générés

Après le build, vous trouverez les installateurs dans :
```
src-tauri/target/release/bundle/nsis/PodcastSync_0.1.0_x64-setup.exe
src-tauri/target/release/bundle/msi/PodcastSync_0.1.0_x64_en-US.msi
```

Uploadez ces fichiers sur la release GitHub v0.1.0.
